# DiceRPG MVP (Android/Kotlin)

MVP Android em **Kotlin** com foco no núcleo do projeto mRPG:

- Chat assíncrono por persona (Eu/Personagem/NPC/Narrador).
- Rolagem de dados inline no chat (`NdM+K`).
- Base preparada para evoluir para campanhas, fichas customizadas, macros e LFP.

## Compatibilidade Android

- `minSdk 21` (Android 5.0) para cobrir o máximo de dispositivos.
- `targetSdk 34`.

## Executar

```bash
cd android-app
./gradlew assembleDebug
```
